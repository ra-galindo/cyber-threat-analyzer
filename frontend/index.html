How This Project Works

This project demonstrates an end-to-end machine learning system for detecting cyber threats in textual data such as phishing emails or malicious messages. The model was created as part of my group project in CPSC 556, Machine Learning Biometric Security, at the University of Calgary. I then later deployed the model and built this frontend as a personal project.

Model

The classifier is based on a fine-tuned BERT transformer model, trained on labeled cybersecurity text to recognize threat categories.

Backend

The model is served through a FastAPI backend deployed on Hugging Face Spaces. The API exposes a simple /analyze endpoint that returns ranked predictions with confidence scores.

Frontend

This frontend is a lightweight, framework-free interface built with vanilla HTML, CSS, and JavaScript. It communicates directly with the deployed API and visualizes prediction confidence in a user-friendly way.

Design Goals

  • Clear separation between model, API, and UI
  • Minimal dependencies and fast load time
  • Production-style deployment using real infrastructure
